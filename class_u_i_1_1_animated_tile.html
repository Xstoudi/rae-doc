Detailed Description

Utility class for Unity.

One of the classes of the "2D Tilemap Extras" library.

This library allows you to easily create animated sprites in the unity editor.

Source of the file : https://github.com/Unity-Technologies/2d-extras/blob/4eff73f468142907629f09b4a88bceb2f0148dc5/Runtime/Tiles/AnimatedTile/AnimatedTile.cs

As the "2D Tilemap Extras" library was not available for the version of Unity used, its script was directly integrated into the project
